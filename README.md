## setup
```bash
yarn global add firebase-tools
firebase login
firebase init
yarn add firebase
```
add File
**./firebaseConfig.js**
```javascript
export default {
  apiKey: "your key",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
  measurementId: ""
}
```

firebase deploy
